Khởi tạo và backup dữ liệu

Dùng khi cài mới hoặc cài lại máy tính: root có thể tạo/cập nhật bảng dữ liệu theo phiên bản hiện tại và lưu đường dẫn thư mục backup trên máy cài server.

Tạo database/user MariaDB

Dùng khi cài lại máy hoặc database/user MariaDB bị mất. Nhập tài khoản quản trị MariaDB để tạo lại database và user ứng dụng theo cấu hình phía trên.

Phục hồi dữ liệu từ máy cũ

Copy file backup SQL từ máy cũ sang máy cài server, sau đó nhập đường dẫn tuyệt đối tới file .sql. Nên tạo database/user MariaDB trước khi phục hồi.

Tạo bảng dữ liệu

Tạo bảng còn thiếu, chạy cập nhật cấu trúc và tạo tài khoản mặc định nếu chưa có. Chức năng này không xóa dữ liệu hiện tại.

Thư mục backup khi cài lại máy

Trình duyệt không đọc được đường dẫn thư mục thật trên máy chủ, vì vậy hãy nhập trực tiếp đường dẫn. Nếu thư mục chưa có, hệ thống sẽ tự tạo.
